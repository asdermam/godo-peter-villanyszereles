# Godó Péter villanyszerelés — demó weboldal

Bemutató (demó) landing weboldal **Godó Péter villanyszerelés** (Erdőkertes, egyéni vállalkozó) számára.

**Élő demó:** https://asdermam.github.io/godo-peter-villanyszereles/

Megbízható villanyszerelés magánszemélyeknek és cégeknek Erdőkertesen és környékén — teljes körű villanyszerelés, új hálózat kiépítése, meglévő bővítése és cseréje, hibakeresés és javítás, valamint villanybojler tisztítás, javítás és csere.

## Technika
- React 18 + Tailwind CSS v4, Vite-tel egyfájlos HTML-be bundle-ölve (`vite-plugin-singlefile`).
- Minden inline: stílus, JS és a logó (base64) — egyetlen `index.html`.
- Sötét „elektromos" téma: éjkék alap + borostyán-sárga (amber) villám akcent + elektromos cián, áramköri (circuit) motívumok.

A részletes build- és tesztnaplóért lásd: [BUILD-LOG.md](./BUILD-LOG.md).

> Ez egy bemutató weboldal. A kapcsolati űrlap demó (nem küld valós üzenetet); a telefon, e-mail, térkép és Facebook linkek élőek.
